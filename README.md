# 新亦源 · GEO 内容工作台

AI 驱动的 SEO 内容生成与多平台分发工具，基于小米 MiMo 大模型。

🔗 **在线访问：** [https://tomatopia.top](https://tomatopia.top)

## 功能

### 📊 数据看板（首页）
- 6 项核心指标卡片：总问题数、已生成母稿、角度覆盖、平台版本、已测试、AI 提及率
- 10 种可视化图表：优先级/搜索意图/内容状态/行业/选题簇分布、内容生产进度环形图、测试摘要、热门卖点 Top5、角度分布、平台覆盖
- 问题×角度覆盖矩阵 — 识别内容空白区
- 3 列响应式网格布局

### 📋 客户提问词库
- 49 个行业问题管理，支持 Excel 导入导出
- 多维度筛选：行业/优先级/搜索意图/状态/选题簇
- 问题列表显示已生成角度数量标签

### 🎯 卖点弹药库
- 14 条核心卖点，按维度分类
- 列表/表格布局，支持 Excel 导入导出

### ✍️ 内容工作台
- AI 生成母稿（7 段式结构：结论先行→原因→场景→数据→风险→建议→引导）
- 母稿写作规范作为唯一 System Prompt 固化
- 5 个核心角度：FAQ 问答、成本拆解、避坑指南、案例故事、对比选型
- 多角度生成 — 同一问题可从不同切入角度生成母稿

### 📡 一稿多发
- 8 个平台专属 Prompt：知乎、百家号、公众号、今日头条、搜狐号、网易号、B站、腾讯新闻
- 各平台版本独立存储，刷新不丢失
- 重新生成按钮 — 已有保存内容时可强制重新生成

### ⚡ 批量生成
- 选题范围 + 模型 + 并发控制
- 支持选择角度批量生成

### 📌 看板
- 任务拖拽管理（未开始/进行中/已发布）
- 可搜索、可多选、提交到「进行中」的模式

### 📄 文稿管理
- 列表展示所有已生成文章
- 8 平台状态标签（知乎/百家号/公众号/今日头条/搜狐号/网易号/B站/腾讯新闻）
- 搜索筛选功能 — 按关键词、平台、状态、角度筛选文章
- 查看编辑弹窗、导出 Word、批量删除

### 🧪 测试记录
- AI 搜索实测数据记录
- 词库与测试记录双向同步

### 📜 更新日志
- 版本迭代记录，时间线布局

### 💾 数据管理
- JSON/Excel 导入导出 + Word 文稿导出
- 数据备份与恢复

## 技术栈

- 前端：原生 HTML + CSS + JavaScript（单页应用）
- 后端：Node.js（API 代理，保护密钥）
- 存储：浏览器 localStorage
- AI：小米 MiMo v2.5 系列模型

## 启动

```bash
npm install
node server.js
# 访问 http://localhost:3010
```

## 配置

服务端 `config.json` 存储 API 密钥（已 gitignore）：

```json
{
  "apiKey": "your-api-key",
  "endpoint": "https://token-plan-sgp.xiaomimimo.com/v1",
  "model": "mimo-v2.5-pro"
}
```

## 部署

### Nginx 反代配置

```nginx
server {
    listen 8080;
    server_name tomatopia.top www.tomatopia.top;

    location / {
        proxy_pass http://127.0.0.1:3010;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### Cloudflare 配置

1. DNS 记录：A 记录指向服务器 IP，开启代理（橙色云朵）
2. Origin Rules：设置源站端口为 8080
3. SSL/TLS 模式：Full
